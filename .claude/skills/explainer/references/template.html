The thing you want to understand

One or two sentences that frame what the reader will walk away knowing — the promise of the page. This template shows every component in the house style; replace the content, keep the system.

Body text sits in a calm 720px column at 18px with airy line-height. Inline accents like a link or a token appear sparingly. The accent color is a spice, not a sauce: a handful of touches per page, never decoration for its own sake.

A section heading in serif

Headings use a system serif for a warm, editorial feel; body and UI use a clean system sans. No web fonts — everything renders offline in a single file.

Note

A callout holds the one thing not to miss in a section. Left accent border, quiet surface. Use note, tip, or warning.

Warning

The warning variant shifts to a warm amber so it reads as caution without shouting.

When there's a sequence: steps

  1. Frame the idea. Steps carry processes and ordered reasoning, each one a small move forward.
  2. Show, don't tell. Prefer an interaction or a concrete example over abstraction.
  3. Land it. Close with the single sentence to remember.

Depth on demand

Hide the proof, the aside, the "but why" behind a collapsible so the main thread stays clean.
